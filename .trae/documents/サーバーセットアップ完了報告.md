# サーバー構築完了

PowerShellの実行ポリシーエラーが発生していましたが、一時的にポリシーをバイパスしてスクリプトを実行し、サーバーのセットアップを完了しました。

## 実施した内容
1.  **サーバーのセットアップ**: `server` フォルダを作成し、Paper 1.21.1 をダウンロードしました。
2.  **EULAの同意**: `eula.txt` を自動生成しました。
3.  **起動スクリプト**: `start.bat` を作成しました。

## 次の手順

以下の手順でプラグインを導入し、サーバーを起動してください。

1.  **プラグインの配置**:
    ビルドした `target/HomesPlugin-1.0-SNAPSHOT.jar` を `server/plugins` フォルダにコピーしてください。
    (※ `plugins` フォルダがない場合は、一度サーバーを起動するか、手動で作成してください)

2.  **サーバーの起動**:
    `server` フォルダ内の `start.bat` をダブルクリックするか、以下のコマンドを実行してください。

    ```powershell
    cd server
    .\start.bat
    ```

これで、ホームプラグインが入った状態でサーバーが起動します。
